# SYJ Voice Assistant

A fully local AI voice assistant for Android Termux.

## Features

- 🎤 Push-to-talk microphone recording
- 📝 Offline speech recognition with whisper.cpp
- 🤖 Local LLM using Ollama
- 🔊 Offline text-to-speech with Piper
- 📱 Designed for Android Termux

## Pipeline

Microphone
→ Whisper.cpp
→ Ollama
→ Piper TTS
→ Audio Playback

## Requirements

- Termux
- Termux:API
- ffmpeg
- whisper.cpp
- Ollama
- Piper
